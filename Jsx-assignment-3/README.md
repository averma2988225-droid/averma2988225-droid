# JSX Assignment 3

A React/JSX assignment project demonstrating core React concepts.

## Features

- **StudentCard** – Displays student information using props
- **CourseList** – Renders a dynamic list of courses using `.map()`
- **Counter** – Demonstrates `useState` hook with increment, decrement, and reset

## Project Structure

```
Jsx-assignment-3/
├── public/
│   └── index.html
├── src/
│   ├── components/
│   │   ├── StudentCard.jsx
│   │   ├── CourseList.jsx
│   │   └── Counter.jsx
│   ├── App.jsx
│   └── index.jsx
├── package.json
└── README.md
```

## Getting Started

```bash
cd Jsx-assignment-3
npm install
npm start
```

Opens on [http://localhost:3000](http://localhost:3000).

## Concepts Covered

- JSX syntax
- Functional components
- Props passing
- Rendering lists with `.map()` and `key`
- `useState` hook
- Inline styles in React
